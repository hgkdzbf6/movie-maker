## ⏰ 定时任务系统

### 任务调度器配置

#### 每小时执行的任务

```typescript
// tasks/scheduler.ts
interface ScheduledTask {
  id: string;
  name: string;
  interval: 'hourly' | 'daily' | 'weekly';
  execute: () => Promise<void>;
  lastExecution?: Date;
  nextExecution?: Date;
}
```

#### 当前定时任务列表

1. **自动保存** (每小时)
   - 自动保存项目到数据库
   - 验证数据完整性
   - 单元测试：自动保存逻辑测试

2. **导出队列处理** (每小时)
   - 处理待导出的任务
   - 更新导出进度
   - 单元测试：队列处理逻辑测试

3. **清理临时文件** (每小时)
   - 清理过期的临时文件
   - 释放磁盘空间
   - 单元测试：文件清理逻辑测试

4. **生成缩略图** (每小时)
   - 为未生成缩略图的视频生成缩略图
   - 更新素材元数据
   - 单元测试：缩略图生成测试

5. **备份数据库** (每小时)
   - 备份 SQLite 数据库
   - 备份到云存储
   - 单元测试：备份逻辑测试
