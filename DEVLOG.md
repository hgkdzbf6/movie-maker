# Remotion 视频编辑器开发日志

## 2026-03-07

### 调研 Remotion Editor Starter

参考了 Remotion 官方的 Editor Starter，调研了以下内容：

#### 技术栈
- **React 19** - 前端框架
- **Remotion 4.0** - 视频引擎
- **React Router v7** - 路由（参考实现）
- **Tailwind CSS v4** - 样式框架
- **React 原生状态管理** - useState + useContext
- **Radix UI** - 无头组件
- **Zustand** - 状态管理（当前项目使用）

#### 核心架构
- 深度嵌套的 Context Provider（性能优化）
- 分离可撤销/不可撤销状态
- 80+ 功能标志控制
- 完整的时间轴功能
- 素材管理系统
- 后端 API 集成

#### 主要功能
1. **Action Row** - 顶部工具栏
   - 绘制纯色块、添加文本、导入素材
   - 撤销/重做、保存项目
   - 下载/加载状态

2. **Timeline** - 时间轴
   - 可拖拽播放头
   - 拖放素材到时间轴
   - 电影条缩略图
   - 自动时长检测
   - 多选/框选
   - 延伸/收缩手柄
   - 音频波形显示
   - 音量控制
   - 淡入/淡出效果
   - 分割项目
   - 隐藏/静音轨道
   - 滚动编辑
   - 播放头跟随
   - 时间轴吸附

3. **Inspector** - 属性面板
   - 组合属性（尺寸、渲染）
   - 来源信息
   - 层对齐
   - 位置/尺寸/裁剪/圆角/透明度控制
   - 字体族/样式/大小
   - 文本对齐/内容/颜色
   - 旋转

4. **后端 API**
   - POST /api/captions - 字幕生成
   - GET /api/font/:name - 字体元数据
   - POST /api/upload - S3 上传
   - POST /api/render - 触发渲染
   - POST /api/progress - 渲染进度
