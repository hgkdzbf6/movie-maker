## 📐 新布局结构

### 1. 整体布局

```
┌─────────────────────────────────────────────────────────────────────┐
│ 顶部导航栏 (56px)                            │
├─────────────┬──────────────────┬──────────────────┬─────────┤
│  左侧边栏   │  中央预览区       │  右侧属性面板   │         │
│  (280px)   │  (Flex-1, ~60%)   │  (320px)         │         │
│             │                  │                  │         │
│             │  - 素材库        │  - 属性面板      │         │
│             │  - 项目设置      │  - 动态切换      │         │
│             │  - 导出选项      │                  │         │
│             │                  │                  │         │
├─────────────┴──────────────────┴──────────────────┴─────────┤
│  底部时间轴 (160px)                           │
└─────────────────────────────────────────────────────────────────────┘
```

### 2. 各区域详情

#### 顶部导航栏 (56px)

**左侧**: 项目名称 + 保存/撤销按钮
- Logo（渐变色背景）
- 项目名称 "Remotion Editor"
- 保存按钮 (Ctrl+S)
- 撤销按钮 (Ctrl+Z)
- 重做按钮 (Ctrl+Shift+Z)
- 未命名项目指示（绿色圆点）

**中间**: 预览分辨率 + 帧率
- 分辨率: "1920×1080 | 30fps"
- 深色背景，圆角标签

**右侧**: 导出 + 设置 + 用户
- 导出按钮（蓝色，带图标）
- 全屏按钮
- 设置按钮
- 用户头像（首字母 Z）

#### 左侧边栏 (280px)

**顶部标签**: 素材库 | 项目 | 导出

**内容区域**:
1. **素材库**
   - 上传区（拖拽上传或点击选择）
   - 分类标签（全部、视频、图片、音频）
   - 素材列表（缩略图 + 文件名 + 时长）

2. **项目**
   - 场景列表（显示所有场景）
   - 支持拖拽排序
   - 复制/删除场景

3. **导出**
   - 格式选择（MP4, GIF, WebM, PNG 序列）
   - 质量（低/中/高）
   - 导出设置（文件名、位置）

#### 中央预览区 (Flex-1, ~60%)

**播放控制区** (56px):
- 播放/暂停按钮（大尺寸，蓝色背景）
- 时间显示（当前时间 / 总时长）
- 场景切换（上一个 / 下一个）
- 分辨率和帧率显示

**视频预览区**:
- 16:9 视频（1920×1080）
- 支持缩放（Fit, 100%, 200%）
- 平移功能
- 黑色背景
- 居中显示

**当前场景信息** (40px):
- 当前场景名称
- 帧范围（开始帧 - 结束帧）

#### 右侧属性面板 (320px)

**标签页**: 属性（单标签）

**动态内容**:
- 未选中：项目全局设置（背景色、默认帧率、分辨率）
- 选中视频/图片：变换、样式、裁剪
- 选中文本：字体、字号、对齐方式、颜色
- 选中音频：音量、淡入淡出、播放速率

#### 底部时间轴 (160px)

**轨道区**:
- 视频轨道（多层叠加，缩略图）
- 音频轨道（波形预览，静音按钮）
- 文本/形状轨道（顶层图层）

**时间标尺**:
- 秒/帧刻度
- 支持缩放（滚轮缩放时间粒度）

**播放头**:
- 红色竖线
- 可拖拽定位

**控制区**:
- 播放模式（循环、单次）
- 标记点（添加章节、导出范围）
- 吸附开关

## 🔧 技术实现

### 响应式设计

**桌面端优先**:
- 最小宽度: 1280px
- 推荐分辨率: 1920×1080

**固定宽度**:
- 左侧边栏: 280px (固定)
- 右侧面板: 320px (固定)
- 中央预览: flex-1 (自适应)

**固定高度**:
- 顶部导航: 56px
- 播放控制: 56px
- 底部时间轴: 160px

### 组件层次

```
EditorPage (h-screen w-screen)
├── Header (h-14)
├── MainContent (flex-1)
│   ├── LeftPanel (w-[280px])
│   ├── PreviewArea (flex-1)
│   └── RightPanel (w-[320px])
└── Timeline (h-40)
```

### 状态管理

**使用 Zustand**:
- 项目状态（scenes, assets, currentFrame）
- UI 状态（activeTab, activePanel, isFullscreen）
- 拖拽状态（draggedScene, isDragging）

## 🔒 已知限制

### 当前限制

1. **响应式**
   - 仅支持桌面端
   - 最小分辨率: 1280px

2. **功能缺失**
   - 关键帧编辑未实现
   - 多轨道支持未实现
   - 转场效果未实现
   - 音频波形未显示

3. **性能**
   - 大量场景时可能有性能问题
   - 视频预览可能卡顿

### 改进方向

1. **性能优化**
